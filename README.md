E-commerce Sales Analysis Using MySQL

Project Summary
“Analyzed a simulated e-commerce dataset of 10,000 orders using MySQL to uncover trends, revenue patterns, customer behavior, and high-value segments. Built views, stored procedures, and advanced queries to simulate real-world business intelligence tasks.”

Skills Demonstrated
SQL (JOINs, GROUP BY, Window Functions, Views, Stored Procedures)
Data Aggregation & Analysis
Business Metrics Calculation (AOV, Revenue, Repeat Customers)
Dashboard-ready Query Preparation
Dataset
CSV file with 10,000 synthetic orders
Columns: order_id, customer_id, product_category, product_price, quantity, order_date, payment_method, city, total_amount
Analysis Highlights
Total Revenue and AOV calculation
Top categories and cities by revenue
Monthly and day-wise sales trends
Repeat customer analysis & cohort analysis
Dashboard-ready queries for key business metrics

How to Run
Upload CSV into MySQL
Execute ecommerce_10000_rows.sql
Queries are ready to run, or call procedures for specific insights
